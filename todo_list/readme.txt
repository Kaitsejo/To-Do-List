1. Installer Wamp
Téléchargez WampServer

2. Configuration de Wamp
Lancez WampServer.
Démarrez les services Apache et MySQL. Assurez-vous que l'icône Wamp dans la barre des tâches est verte.

3. Créer la base de données MySQL
Accédez à phpMyAdmin en ouvrant votre navigateur et en visitant http://localhost/phpmyadmin.
Créez la base de données et la table à l'aide du fichier todo_list.sql

4. Configurer le projet dans Wamp
Créez un répertoire de projet :
Naviguez vers le répertoire www de WampServer (généralement C:\wamp64\www).
Déplacer le dossier to_do_list dans le dossier www

5. Lancer le projet
Ouvrez votre navigateur et accédez à http://localhost/todo_list/index.php.
Testez votre application en ajoutant, marquant comme accomplies et supprimant des tâches.
