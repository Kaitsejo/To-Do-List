En français

Installer WampServer

Téléchargez WampServer à partir du site officiel.
Suivez les instructions d'installation pour installer WampServer sur votre ordinateur.
Configuration de WampServer

Lancez WampServer après l'avoir installé.
Assurez-vous que les services Apache et MySQL sont démarrés. Vous pouvez vérifier cela en vous assurant que l'icône Wamp dans la barre des tâches est verte.
Créer la base de données MySQL

Ouvrez votre navigateur web.
Accédez à phpMyAdmin en visitant http://localhost/phpmyadmin.
Connectez-vous à phpMyAdmin si nécessaire.
Importez le fichier todo_list.sql pour créer la base de données et les tables nécessaires.
Configurer le projet dans WampServer

Créez un répertoire pour votre projet. Par exemple, créez un dossier nommé todo_list.
Naviguez vers le répertoire www de WampServer. Habituellement, le chemin est C:\wamp64\www sur Windows.
Déplacez le dossier todo_list que vous avez créé dans le répertoire www.
Lancer le projet

Ouvrez votre navigateur web.
Accédez à votre projet en visitant http://localhost/todo_list/index.php (assurez-vous que le nom du dossier et le nom du fichier index.php correspondent à votre configuration).
Testez votre application todo list en ajoutant, en marquant comme accomplies et en supprimant des tâches.

In English 

Install WampServer

Download WampServer from the official website.
Follow the installation instructions to install WampServer on your computer.
Configure WampServer

Launch WampServer after installation.
Start Apache and MySQL services. Ensure the Wamp icon in the taskbar turns green.
Create MySQL Database

Open your web browser.
Access phpMyAdmin by visiting http://localhost/phpmyadmin.
Log in to phpMyAdmin if required.
Import the todo_list.sql file to create the necessary database and tables.
Set Up Project in WampServer

Create a project directory. For example, create a folder named todo_list.
Navigate to the www directory of WampServer. Typically, it's located at C:\wamp64\www on Windows.
Move the todo_list folder you created into the www directory.
Launch the Project

Open your web browser.
Access your project by visiting http://localhost/todo_list/index.php (make sure the folder name and index.php file name match your configuration).
Test your todo list application by adding, marking as completed, and deleting tasks.
